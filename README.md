# mymessenger

#### [목표] 
- 특정 주소를 구독한 사용자에게 
- 특정 주소 주변 공적마스크 재고/판매처를 찾아
- 실시간 메시지 전송

#### [이슈]

1. 카카오 oauth2 연동

2. DB연동 및 계정 기록

3. 계정별 주소 구독

4. 구독 주소 주변 /storesByAddr/json 주기적으로 요청하는 배치서버 개발
    - 구독 주소 계정별 입력 UI 작업
    - 트랜잭션 설정 
    - 계정별 구독 주소 DB 관리
    - 배치 틱마다 이전 틱의 데이터(마스크 재고) DB 관리해 비교

5. 메신저 연동
 - 카카오톡 유력 (단, 비용문제로 다른 메신저 변경가능하도록 추상화해서 작업할것)
